# Парсер статей Habr

Парсер для поиска статей на Habr по ключевым словам.

## Задание

Скрипт парсит главную страницу Habr и ищет статьи, содержащие заданные ключевые слова в тексте.

## Установка  зависимостей

```bash
pip install requests beautifulsoup4
```

## Функционал

- **parsehabr.py - Основной режим**: парсит превью статей на главной странице
- **parsehabfull.py - Расширенный режим**: переходит на каждую статью и парсит полный текст

## Ключевые слова

По умолчанию ищет: `дизайн`, `фото`, `web`, `python`

Список можно изменить в переменной `KEYWORDS` в коде.

## Вывод

Для каждой найденной статьи выводится:

- Дата публикации
- Заголовок
- Ссылка
- Найденные ключевые слова
